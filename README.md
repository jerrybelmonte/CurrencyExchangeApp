# Currency Exchange App
Find out the exhange rate of 1 USD in other major currencies.

## Objective
Get some experience working with flutter and learn advanced state management, advanced http requests, JSON parsing, ListTile, CircleAvatar, BorderRadius, and advanced Dart coding to filter results.
